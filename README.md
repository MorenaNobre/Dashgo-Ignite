## Sobre o Projeto

<br>

<p>O Dashgo é a interface de um painel administrativo contendo o fluxo de autenticação em seu formulário, listagem e cadastro de usuários.</p>

<p>Nessa aplicação foram utilizadas as seguintes tecnologia e conceitos:</a>

* React;
* Typescript;
* MirageJS;
* Hooks;
* Chakra UI;
* Apexcharts;
* React-Query;
* Axios;
* Context;
